This is the API

appliancesImplGet

Get the list of all the appliance implementations registered. No authentification is required.

URL

http://127.0.0.1:8003/appliances_impl/

HTTP Method

GET

Response Type

List

Parameters

appliancesImplNameDelete

Delete an already existing appliance implementation. Authentification is required.

URL

http://127.0.0.1:8003/appliances_impl/{name}/

HTTP Method

DELETE

Response Type

Parameters

  • Path: name String

    Name of the appliance implementation

appliancesImplNameGet

Get a single appliance implementation. Authentification is not required.

URL

http://127.0.0.1:8003/appliances_impl/{name}/

HTTP Method

GET

Response Type

ApplianceImplementation

Parameters

  • Path: name String

    Name of the appliance implementation

appliancesImplNamePatch

Modify an already existing appliance implementation. Authentification is required.

URL

http://127.0.0.1:8003/appliances_impl/{name}/

HTTP Method

PATCH

Response Type

ApplianceImplementation

Parameters

    Name of the appliance implementation

    Appliance implementation
