This is the API

scriptsApplianceActionGet

Get the script corresponding to an appliance and an action. Authentification is not required.

URL

http://127.0.0.1:8003/scripts/{appliance}/{action}/

HTTP Method

GET

Response Type

Script

Parameters

  • Path: appliance String

    Name of the appliance

  • Path: action String

    Name of the action

scriptsGet

Get the list of all the scripts registered. No authentification is required.

URL

http://127.0.0.1:8003/scripts/

HTTP Method

GET

Response Type

List

Parameters
