Cluster

  • id : Integer
    Unique ID of the cluster
  • name : String
    Name given to the cluster
  • uuid : String
    Internal UUID given to the cluster
  • hints : String
    hints that will be used to select the infrastructure that will host cluster's nodes
  • credential : String
    hints that will be used to select the infrastructure that will host cluster's nodes
  • public_key : String
    Public key used to identify nodes of the cluster
  • status : String
    Status of the cluster
  • hosts_ids : List
    IDs of nodes of the cluster (can be null)
  • targeted_slaves_count : Integer
    targeted number of slaves in the cluster
  • current_slaves_count : Integer
    current number of slaves in the cluster
  • hosts_ips : List
    IP addresses of nodes of the cluster (can be null)
  • master_node_id : Integer
    ID of the master node of the cluster (can be null)
  • master_node_ip : String
    IP address of the master node of the cluster (can be null)
  • user_id : Integer
    ID of the user that created the cluster
  • appliance : String
    Appliance of the cluster
  • appliance_impl : String
    Appliance of the cluster
