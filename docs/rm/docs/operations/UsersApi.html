This is the API

apiTokenAuthPost

Get the authentification token for a user by providing his username and password. No authentification is required; however the credentials of the user are part of the request.

URL

http://127.0.0.1:8002/api-token-auth/

HTTP Method

POST

Response Type

TokenResp

Parameters

  • Body: data Credentials(Credentials)

    User credentials

Response Type

PublicKey

    ID of the user
